层和盒子 Layouts and box

Layout

层(Layout)用于将页面划分为几个基本的布局，分别是layout-header,layout-sidebar,layout-menu,layout-main。

box

盒子(box)是在同一页面中将内容进一步划分区域的容器，比如下文中layout-header,layout-menu等说明的区域装载体。

响应式

layout-sidebar根据屏幕大小显示为可见和隐藏状态,同时其余layouts也会调整至最佳宽度。

Layout-header
layout-header用于显示APP的头部。
Layout-menu
layout-menu用于显示主菜单，固定在底部。
Layout-sidebar
layout-sidebar用于显示侧栏，宽屏下默认显示，窄屏下默认隐藏。
Layout-main
layout-main用于显示页面主题部分。
